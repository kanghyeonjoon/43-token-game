AI 자동 자막 (음성 인식)

영상의 음성을 인식해 단어 타임스탬프가 달린 클립을 자동 생성합니다. 기존 클립은 대체됩니다(Ctrl+Z로 복구 가능).
처음 실행 시 모델을 내려받으며(인터넷 필요) 이후엔 브라우저에 캐시되어 오프라인으로도 동작합니다. 인식은 CPU(WASM)에서 실행되어 영상 길이에 따라 수 분이 걸릴 수 있습니다. 한국어는 base 이상을 권장합니다.
음성은 16kHz 모노 WAV로 변환되어 입력한 API 주소로만 전송됩니다. 키는 이 브라우저의 localStorage에만 저장됩니다. Groq 등 OpenAI 호환 서버도 주소/모델만 바꾸면 사용할 수 있습니다.

사용법

Vrew 방식의 "텍스트로 영상 편집" 클론 프로토타입입니다. 모든 처리가 브라우저 안에서만 이루어집니다(업로드 없음).
1. 불러오기 — 파일 탭에서 영상(mp4/webm)을 불러온 뒤, SRT 자막을 불러오거나 대본을 붙여넣으면 단어 단위 클립이 생성됩니다.
1-1. AI 자동 자막 — 자막 탭에서 음성 인식으로 클립을 자동 생성할 수 있습니다. 브라우저 로컬(Whisper, 무료·키 불필요) 또는 OpenAI 호환 API 중 선택하세요.
2. 단어 컷 편집 — 단어 칩 클릭으로 선택(Shift 범위, Ctrl 다중), Delete 또는 편집 탭 ✂️로 컷. 컷한 단어의 영상 구간은 재생·내보내기에서 건너뜁니다. 다시 선택해 ♻️ 복원.
3. 자막 수정 — 클립 아래 💬 줄을 클릭해 오타를 고치면 단어 칩이 다시 배분됩니다.
4. 클립 다듬기 — 단어를 선택하고 🧩 분할하면 그 단어부터 새 클립, 🔗 병합은 아래 클립과 합칩니다.
5. 무음 컷 — 영상을 불러온 상태에서 🔇를 누르면 오디오를 분석해 0.4초 이상의 무음 구간을 자동으로 컷합니다.
6. 내보내기 — SRT(컷 반영 타임라인), 편집본 WebM(실시간 렌더), 편집툴 XML(xmeml — Premiere Pro·DaVinci Resolve 호환), 컷 데이터 JSON, 프로젝트 저장을 지원합니다.
6-1. XML 활용법 — XML 파일을 원본 영상과 같은 폴더에 두고 프리미어 파일→가져오기(리졸브는 File→Import→Timeline)로 열면 컷 편집이 시퀀스로 복원됩니다. 자막 문장은 타임라인 마커로 들어가며, 자막 트랙이 필요하면 SRT를 함께 가져오세요.
단축키 — Space 재생 · Delete 컷 · Ctrl+Z 실행취소 · Ctrl+Y 다시실행
