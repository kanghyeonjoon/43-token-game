# 케어루프 CRM — 자동 문자 발송 연동 가이드 (솔라피)

[발송하기] 버튼이 환자 휴대폰으로 **실제 문자를 바로 발송**하게 만드는 절차입니다.
연동 전에도 앱은 "문안 복사 + 발송 기록"으로 동작하므로, 준비되는 대로 켜면 됩니다.

## 비용 (선불 충전, 월 기본료 없음)

| 종류 | 건당 비용(약) | 비고 |
|---|---|---|
| SMS (90바이트 이하) | ~9원 | 짧은 안내 |
| LMS (장문) | ~27원 | 케어루프 템플릿 대부분 |
| 카카오 알림톡 | ~8원 | 카카오 채널 + 템플릿 심사 후 (아래 확장) |

앱이 문안 길이에 따라 SMS/LMS를 자동 선택합니다. **야간(21:00~08:00)은 자동으로 다음날 오전 9시 예약 발송**됩니다(정보통신망법 야간 광고 제한 준수).

## 1단계 — 솔라피 가입 · 발신번호 등록 (하루 걸림, 먼저 시작)

1. [solapi.com](https://solapi.com) 회원가입 → 콘솔 접속
2. **발신번호 등록**: 콘솔 → 발신번호 관리 → 병원 대표번호 등록
   - 통신서비스 가입증명원(병원 명의) 업로드 → **심사 약 1영업일**
3. **API 키 발급**: 콘솔 → API Key 관리 → 새 API Key 생성
   - `API Key`와 `API Secret` 두 값을 복사해 둡니다 (Secret은 이때만 보임)
4. 충전: 콘솔 → 충전하기 (1만원이면 LMS 약 370건)

## 2단계 — Supabase에 발송 함수 만들기 (5분, 코드 붙여넣기)

⚠ API Secret을 앱(브라우저)에 넣으면 안 되기 때문에, Supabase의 무료 **Edge Function**이 발송을 대행합니다.

1. Supabase 대시보드 → 왼쪽 메뉴 **Edge Functions** → **Deploy a new function** (또는 Create function → via Editor)
2. 함수 이름: **`send-sms`** (정확히 이 이름)
3. 에디터에 저장소의 **`db/edge-send-sms.ts`** 내용 전체를 붙여넣기 → **Deploy**

## 3단계 — 시크릿 3개 설정

Edge Functions → **Secrets** (또는 Settings → Edge Functions → Secrets):

| 이름 | 값 |
|---|---|
| `SOLAPI_KEY` | 1단계에서 발급한 API Key |
| `SOLAPI_SECRET` | 1단계에서 발급한 API Secret |
| `SOLAPI_SENDER` | 심사 통과한 발신번호 (숫자만, 예: 05071234567) |

## 4단계 — 앱에서 테스트

1. 케어루프 접속(로그인) → **병원관리 → 환경설정 → 자동 발송 (솔라피)** 카드
2. **[연동 테스트]** → 원장님 휴대폰 번호 입력 → 시험 문자 1건 수신 확인
3. 성공하면 끝 — 이제 **개별 발송 → [발송하기]** 가 환자에게 실제 문자로 나갑니다
   (환자 전화번호가 등록돼 있어야 해요 — 환자 등록 창에 전화번호 칸이 있습니다)

## 문제 해결

- **"솔라피 시크릿이 설정되지 않았습니다"** → 3단계 Secrets 이름 철자 확인 (SOLAPI_KEY/SOLAPI_SECRET/SOLAPI_SENDER)
- **"발신번호가 등록되지 않았습니다" 류 솔라피 오류** → 1단계 발신번호 심사가 아직 안 끝난 상태
- **테스트 실패 + 함수 404** → 2단계 함수 이름이 `send-sms`인지 확인
- 자동 발송이 실패하면 앱이 자동으로 "문안 복사 + 기록" 모드로 폴백하므로 진료는 멈추지 않습니다

## 확장 — 카카오 알림톡 (선택, 더 저렴하고 도달률 높음)

1. 카카오 비즈니스에서 **카카오톡 채널** 개설 + 비즈니스 인증
2. 솔라피 콘솔 → 카카오 연동 → 채널 연결 → **알림톡 템플릿 등록·심사** (케어루프의 `docs/CRM-메시지-템플릿.md` 문안 사용, 심사 2~3일)
3. 심사 통과 후 Edge Function을 알림톡 발송(카카오 옵션)으로 확장 — 이 단계가 되면 말씀해 주세요, 함수 코드를 업데이트해 드립니다
