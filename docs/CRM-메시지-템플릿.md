# 경희나비솔 한의원 CRM 발송 메시지 템플릿 (v1.0)

> `CRM-대시보드-기획서.md`(v3.0)의 12개 자동화 시나리오에 대응하는 환자 발송 문안 전체.
> 알림톡(정보성)은 카카오 심사 제출 가능한 형식으로, 광고성 메시지는 친구톡 + LMS 폴백 2벌로 작성했다.

---

## 0. 공통 규칙

### 0.1 발신 명의와 톤
- **원장 1인칭, 따뜻하되 담백한 존댓말.** 과장·재촉 없이, 환자가 "관리받고 있다"고 느끼는 문장.
- 원장 성함·연락처 등은 전부 변수 처리: `#{원장명}`, `#{한의원전화}`.
- 관여도 등급(상/중/하)은 **어떤 문구에도 노출 금지** (기획서 8.2).

### 0.2 변수 정의 (카카오 알림톡 규격 `#{변수명}`)

| 변수 | 내용 | 원천 |
|---|---|---|
| `#{환자명}` | 데스크 확정 표기 이름 | DB1 |
| `#{원장명}` / `#{한의원전화}` | 발신 정보 | 고정값 |
| `#{예약일시}` | 예: 7월 12일(토) 오전 10:30 | DB2 |
| `#{주증상명}` `#{다른증상명}` `#{걱정영역}` | 택소노미 표기 그대로 | DB1/DB2 |
| `#{처방유형}` `#{복용횟수}` `#{복용시점}` `#{주의사항}` `#{남은일수}` | 복약 시퀀스용 | DB3 |
| `#{증상팁문구}` `#{걱정팁문구}` | Part C에서 삽입 | 콘텐츠 라이브러리 |
| `#{환자토큰}` | 클릭 추적용 난수 토큰 | DB1 (발송 시 생성) |

### 0.3 버튼·링크 규칙 (기획서 4.2)
모든 버튼 URL은 Make 웹훅 리다이렉트 패턴을 사용한다. 전화번호 원문 삽입 금지.

```
https://hook.make.com/{웹훅ID}?p=#{환자토큰}&c={콘텐츠태그}&a={액션}
액션 값: click(링크 클릭 +1) / positive(도움이 됐어요 +2) / good·neutral·bad(D+5 응답) / reserve(예약)
```

### 0.4 광고성 메시지 필수 요소 (기획서 8.3)
- 본문 첫 줄: `(광고) 경희나비솔한의원`
- **LMS 버전** 말미: `무료수신거부 080-###-####`
- **친구톡 버전** 말미: `이 메시지를 원치 않으시면 채널 차단으로 수신을 거부하실 수 있습니다.`
- 야간(21:00~08:00) 발송 차단은 시스템(공통 가드 4번)이 처리 — 문안과 무관.
- 건강 정보 문구에는 효능 단정 표현 금지("낫습니다" ✗ → "도움이 됩니다" ✓), 일반 정보 고지 1줄 포함.

### 0.5 민감 병명 완곡 표현 (기획서 10.4 — 환자 노출 문구 전체 적용)

| 내부 관리 용어 | 환자 노출 표기 |
|---|---|
| 중풍(뇌졸중) 걱정 | **혈관 건강** |
| 치매 걱정 | **기억력·뇌 건강** |
| 암 걱정 | **정기 검진** |

### 0.6 템플릿 코드 규칙
`KNB-{분류}-{번호}` — INFO(알림톡 정보성) / AD(친구톡·LMS 광고성) / OPS(내부 알림). 심사·관리 대장(Part E)과 Make 시나리오가 이 코드를 참조한다.

## Part C. 콘텐츠 문안 — 증상별 삽입 문구 + 1분 영상 스크립트 개요

> `#{증상팁문구}` / `#{걱정팁문구}` 슬롯에 삽입. 콘텐츠 태그는 부록 B 택소노미와 글자 단위 일치.
> 영상 공통 구성: ① 인사·공감(10초) ② 원인 한 줄(10초) ③ 생활 관리 2가지 시연(30초) ④ 주의 신호·마무리(10초).

### C-1. 주증상 톱10 (초기 케어용 — 태그 = 택소노미 표기)

**요통**
> 오래 앉았다 일어날 때 뻐근하다면, 30분마다 한 번 일어나 가볍게 허리를 펴 주세요. 주무실 때 무릎 아래에 얇은 베개를 받치면 허리 부담이 줄어듭니다. 통증이 심한 시기에는 무리한 스트레칭보다 온찜질이 먼저입니다.

영상: 앉은 자세에서 허리가 받는 부담 설명 → 30분 기립 습관 + 무릎 베개 시연 → "다리 저림이 함께 오면 바로 알려주세요."

**목·어깨 통증**
> 모니터 상단을 눈높이에 맞추고, 한 시간에 한 번 턱을 뒤로 당기는 동작을 10회 해 보세요. 스마트폰은 가슴이 아닌 눈높이로 드는 것이 기본입니다. 뭉침이 심한 날은 따뜻한 샤워 후 어깨로 크게 원을 그리는 운동이 도움이 됩니다.

영상: 거북목 자세 비교 → 턱 당기기(친 턱) 시연 → 어깨 원 그리기 → "팔 저림 동반 시 내원 안내."

**무릎 통증**
> 계단을 내려갈 때 아프다면 당분간 내리막·계단을 줄이시고, 의자에 앉아 다리를 천천히 펴 올리는 운동으로 허벅지 근육을 지켜주세요. 허벅지 근육이 무릎을 지키는 가장 좋은 보호대입니다.

영상: 무릎에 실리는 하중 설명 → 앉아서 다리 펴기 10회 시연 → 걷기 강도 조절 팁.

**발목염좌**
> 다친 직후 이틀은 무리하게 돌리지 말고, 앉아 계실 때 발목을 심장보다 높게 올려 두세요. 부기가 가라앉기 시작하면 수건을 발바닥에 걸고 천천히 당기는 스트레칭이 회복을 돕습니다. '이제 괜찮다' 싶은 2~3주 차에 재발이 가장 잦으니 그때가 더 조심할 때입니다.

영상: 초기 거상 자세 → 수건 스트레칭 시연 → 재발 주의 구간 설명.

**두통·어지럼**
> 두통이 잦다면 물 섭취량과 수면 시간을 먼저 점검해 보세요. 갑자기 일어날 때 어지럽다면, 눕거나 앉은 자리에서 30초 머문 뒤 일어나는 습관이 도움이 됩니다. 목 뒤가 뻣뻣하면서 두통이 올 때는 따뜻한 찜질을 해 보세요.

영상: 두통 유형 한 줄 구분 → 기상 30초 규칙 → 목 찜질 시연 → "갑작스럽고 심한 두통은 바로 진료가 필요합니다."

**소화불량**
> 식사는 8분 이상 천천히, 물은 식사 중보다 식간에 드시는 것이 좋습니다. 저녁은 잠들기 3시간 전에 마치고, 식후 10분 가벼운 산책이 소화를 돕습니다.

영상: 빨리 먹는 습관과 더부룩함의 관계 → 식간 수분 팁 → 식후 산책 권유.

**불면**
> 잠들기 1시간 전에는 화면을 멀리하고 조명을 한 단계 낮춰 보세요. 침대에서 20분 넘게 뒤척이면 일단 나와서 단조로운 책을 읽다가 다시 눕는 편이 오히려 빠릅니다. 낮잠은 20분 이내로 줄여 주세요.

영상: 수면 리듬 설명 → 20분 규칙 시연 → 낮잠·카페인 팁.

**피로·번아웃**
> 오후를 커피로 버티는 것보다, 점심 후 10분 햇볕 산책이 회복에 더 좋습니다. '피곤한데 잠은 안 오는' 상태라면 몸의 긴장부터 풀어야 합니다 — 자기 전 따뜻한 족욕 10분을 권합니다.

영상: 피로와 긴장의 관계 → 햇볕 산책 → 족욕 시연 → "2주 이상 지속되면 함께 원인을 찾아봅시다."

**손목·팔꿈치 통증**
> 손목이 아플 때는 손목 자체보다 팔꿈치 아래 근육을 풀어주는 것이 효과적입니다. 마우스는 손목이 꺾이지 않게 팔 전체로 움직이고, 한 시간에 한 번 손가락을 쫙 펴는 스트레칭을 해 주세요.

영상: 전완근과 손목 통증의 관계 → 전완 마사지 시연 → 손가락 펴기 스트레칭.

**좌골신경통**
> 엉덩이에서 다리로 뻗치는 통증이 있을 때는 딱딱한 바닥에 오래 앉는 것을 피하고, 서 있는 시간을 틈틈이 섞어 주세요. 무릎을 가슴으로 당기는 스트레칭을 아침저녁 10회씩, 통증이 없는 범위에서만 하세요.

영상: 뻗치는 통증의 경로 설명 → 무릎 당기기 시연 → "발에 힘이 빠지는 느낌이 들면 바로 알려주세요."

### C-2. 걱정증상 톱5 (후기 케어용 — 완곡 표기)

**혈관 건강** *(내부: 중풍 걱정)*
> 혈관은 조용히 나이 듭니다. 아침에 일어나 갑자기 찬 공기를 쐬는 것을 피하고, 짠 음식을 반 숟갈 줄이는 것부터가 시작입니다. 혈압을 정기적으로 확인하는 습관이 가장 든든한 대비입니다.

**기억력·뇌 건강** *(내부: 치매 걱정)*
> 뇌 건강의 기본은 잠·걷기·사람입니다. 하루 30분 걷기, 7시간 수면, 일주일에 한 번 이상 대화가 있는 모임 — 이 세 가지가 가장 검증된 관리법입니다.

**관절 퇴행**
> 관절은 아껴 쓰는 것보다 '바르게 쓰는 것'이 오래 갑니다. 근육이 관절을 지키는 갑옷이니, 통증 없는 범위의 근력 운동을 꾸준히 이어가 주세요.

**대사 건강** *(내부: 당뇨·혈압 걱정)*
> 식후 15분 걷기만으로도 혈당 관리에 도움이 됩니다. 야식과 단 음료를 줄이는 것부터 시작해 보세요. 작은 습관이 숫자를 바꿉니다.

**정기 검진** *(내부: 암 걱정)*
> 가장 확실한 예방은 제때 확인하는 것입니다. 나이와 가족력에 맞는 검진 주기를 지키는 것만으로 큰 걱정의 대부분은 미리 대비할 수 있습니다. 검진 일정 잡기가 막막하시면 다음 내원 때 함께 정리해 드리겠습니다.
