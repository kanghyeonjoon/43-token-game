ℹ 사용 순서

① 채널 브리프(=대표 가이드라인) 입력 → ② [🗣 제작 회의] — AI 직원 5명이 서로의 발언에 반응하며 방향을 합의하고 제작 방향 합의서를 작성 (합의서는 직접 고칠 수 있고, '미결 사항'은 대표가 결정) → ③ [🚀 전체 실행] — 전 단계가 합의서를 기준으로 제작되어 재작업이 줄어듭니다 → ④ 산출물 수정은 '수정 지시'로, 방향 자체가 바뀌면 회의를 다시 → ⑤ 마지막 [📤 업로드 세팅] 시트대로 게시.
데모 모드에서는 실제 AI 호출 없이 예시 산출물로 흐름을 체험합니다. ⚙ 설정에서 Anthropic API 키를 넣으면 브리프 기반 실제 제작으로 전환됩니다.

⚙ 설정 — AI 연결

Anthropic Claude API로 동작합니다 · 키 발급: console.anthropic.com → API Keys
미연결 — 데모 모드
⚠ 키는 이 브라우저(localStorage)에만 저장되고 Anthropic 외 어디로도 전송되지 않습니다. 본인 PC에서만 사용하세요 — 파일을 팀에 공유하거나 웹에 올릴 때는 키를 삭제하고, 서버 프록시(예: Supabase Edge Function) 방식으로 전환해야 합니다. Anthropic 콘솔에서 월 사용 한도(Spend limit)를 걸어두는 것을 권장합니다.
